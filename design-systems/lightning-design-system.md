## 视觉主题与氛围

这个界面说的是"企业的信任"。每个表面、每种颜色、每次交互都为了帮助专业人士更快地推进工作流。设计借鉴了仪表盘和数据表格的语言——在那里，清晰是生产力，而非偏好。

视觉调性是"企业的乐观"。蓝色（#0176D3）承载着自信握手的能量——它说"我们能成交"。温暖的中性色让仪表盘在12小时轮班中保持可读。4px圆角说"专业但不僵化"。信息密度高因为CRM用户扫视而非阅读——每个像素必须传递数据或启用操作。

界面应该像一个信赖的同事：随时待命、从不妨碍、永不意外。

## 排版规则

字体：**Salesforce Sans**、Arial、sans-serif。一种专为CRM界面特定需求设计的自定义字体——高信息密度、长时间阅读、小尺寸仍需保持清晰。

| 级别 | 字号 | 字重 | 为什么 |
|------|------|------|--------|
| 展示 | 44px | 300 | "这是英雄声明。" 细字重在展示尺寸读起来精致而非攻击。 |
| 标题大 | 32px | 300 | "这是页面标题。" 细字重为优雅。 |
| 标题中 | 24px | 300 | "这是主要分节。" |
| 标题小 | 18px | 400 | "这是一个话题。" 从细到常规的过渡标记着从展示到功能的转变。 |
| 题目 | 16px | 700 | "这是强调标签。" 小尺寸粗体=密集数据视图中的可扫视性。 |
| 正文 | 14px | 400 | 默认。14px在1.5行高下=CRM密度中舒适阅读。 |
| 说明 | 12px | 400 | 元数据。本系统的下限。 |

字阶在大尺寸使用细字重（300），在功能标签使用粗体（700）。这种对比创造视觉节奏——展示元素呼吸，功能元素锚定。

## 组件样式

### 按钮（品牌）
- 背景：#0176D3——"自信的行动"
- 高度：32px默认，36px中
- 圆角：4px——专业但不僵化
- 字体：14px，字重400——常规字重因为蓝色提供足够强调
- 悬停：#005FB2——"我看到你了"
- 焦点：box-shadow 0 0 3px #0070D2——柔和的蓝光
- 按下：#00396B——视觉压缩
- 禁用：不透明度0.5

### 按钮——中性
- 背景：白色，边框：1px实线#DDDBDA，文字：#0176D3
- 用于不需要视觉重量的次要操作

### 卡片
- 背景：白色
- 边框：1px实线#E5E5E5
- 圆角：4px——与按钮语言一致
- 阴影：0 2px 2px rgba(0,0,0,0.1)

### 输入框
- 高度：32px
- 圆角：4px
- 焦点：border-color #1589EE, box-shadow 0 0 3px #0070D2
- 错误：border-color #EA001E

### 数据表格
- 表头：#F3F3F3背景
- 行高：36px——紧凑以利扫视密度
- 悬停：#F3F3F3

## 布局原则

- 基础单位：4px (0.25rem)
- 间距比例：4、8、12、16、20、24、32、48、64
- 网格：12列
- 最大内容宽度：1280px
- 圆角比例：4px（默认）→ 8px（中）→ 12px（大）→ 50%（圆形）

## 深度与高度

| 级别 | 用途 | 阴影 | 为什么 |
|------|------|------|--------|
| 1 | 卡片 | 0 2px 2px rgba(0,0,0,0.10) | 微妙抬升。卡片在地板之上但不引人注意。 |
| 2 | 下拉菜单 | 0 2px 3px rgba(0,0,0,0.15) | 临时元素需要更多存在感。 |
| 3 | 弹窗 | 0 2px 4px rgba(0,0,0,0.20) | 弹窗占有用户的注意力。 |

## 宜与忌

宜：
- 专门使用Salesforce Sans。它为本界面而设计。
- 保持高信息密度。CRM用户扫视而非阅读。
- 使用4px圆角作为默认——它是"专业但亲和"的签名。
- 使用12列网格保持布局一致。

忌：
- 不要用品牌蓝做装饰背景。用brand-1（#CCE6FF）代替。
- 每页不要超过3级标题。
- 不要将红色（#EA001E）用于非错误或破坏性操作。
- 不要覆盖36px表格行高——它为扫视密度优化。
- 不要用自定义字体替代Salesforce Sans。

## 响应式行为

| 断点 | 宽度 | 发生什么 |
|------|------|----------|
| 小 | < 768px | 单列，紧凑导航 |
| 中 | 768-1024px | 2列，可折叠侧边栏 |
| 大 | ≥ 1024px | 完整布局，常驻侧边栏 |

- 最小触摸目标：44x44px
- 字号永远不低于12px
